MLB City Match - Team Logo Images
==================================

Place team logo images here with the following naming convention:

  {TEAM_ABBR}.png

Example:
  LAD.png  (Dodgers)
  NYY.png  (Yankees)
  BOS.png  (Red Sox)

Full list of abbreviations:
  AL East:    NYY, BOS, BAL, TB, TOR
  AL Central: CLE, MIN, CWS, DET, KC
  AL West:    HOU, SEA, LAA, OAK, TEX
  NL East:    ATL, NYM, PHI, MIA, WSH
  NL Central: CHC, STL, MIL, CIN, PIT
  NL West:    LAD, SD, SF, ARI, COL

Notes:
- PNG format recommended (transparent background works best)
- Square images work best (they'll be displayed in circles)
- If an image is missing, the text abbreviation is shown instead
- These images are for personal use only and are not included in the repository
